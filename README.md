# Rental DApp — Landlord & Tenant Agreement System

A decentralized application (DApp) for managing rental agreements
between landlords and tenants using Ethereum blockchain.

## Tech Stack
- **Frontend:** Next.js + React + Tailwind CSS
- **Backend:** Node.js + Express + MongoDB
- **Blockchain:** Solidity + Hardhat + Ethereum Sepolia Testnet
- **Wallet:** MetaMask + ethers.js

## Project Structure
- `/frontend` - Next.js application
- `/backend` - Express REST API
- `/blockchain` - Smart contracts
